## 使用方式

**当你有一个新的截图或页面描述时：**

在 Claude Code 中说：
```
这是 [模块名] 的 [页面/功能名]，功能是 [简短描述]
[截图或描述]
[可选：已知问题或注意事项]
```

我会：
1. 自动保存截图到 `/tmp/screenshots/`
2. 更新这个索引文件，加上时间戳和功能说明
3. 记住这个关联，下次生成 PRD 时自动引入
