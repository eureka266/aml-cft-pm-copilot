# \[20260114\] 用户注册强引导 & 结果分享页优化

# 目标

*   提升 visitor → 注册用户转化率
    

*   明确核心能力（Screen）的使用门槛
    

*   提升结果分享页的 SEO 价值
    

#### 需求范围

| 需求模块 | 说明 |
| --- | --- |
| 注册 / 登录强引导 | *   新增 Google 注册 / 登录<br>    <br>*   在用户进入平台早期即强引导注册 / 登录<br>    <br>*   统一账号体系：同一 email 对应唯一用户 |
| Screen 使用门槛调整 | *   **移除 Lite Scan 模式**<br>    <br>*   Screen 仅对已注册 / 登录用户开放<br>    <br>*   用量耗尽用户再次 Screen 时：<br>    <br>    *   明确提示升级 / 购买点数 |
| 结果分享页优化 | *   展示完整扫描结果，不再遮挡<br>    <br>*   引导用户重新扫描查看最新结果<br>    <br>*   支持自动化生成结果分享页<br>    <br>*   用户手动创建结果分享页 |

    *   无对应产品权限时，正常提示（403）
        

#### Sign up（仅 Compliance）

*   Compliance 的 Sign up 页面：
    
    *   增加 `Continue with Google`
        
    *   参加 [Lovable](https://lovable.dev/)：![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqrebXPvmKvnak/img/fc1b7bed-8cee-4fa4-adc2-c2ce96524d4e.png)

## Compliance 注册强引导流程

### 入口 1：落地页搜索框进入

#### 已登录用户

*   **有用量（**和当前保持一致**）**
    
    *   直接执行 Screen
        
    *   进入地址 / 交易详情页
        
*   **无用量**：
    
    *   展示升级/购买弹窗
        
    *   购买完成后自动执行 Screen 并进入详情页
        

#### 未登录用户

*   用户输入地址 / 交易并点击 Screen 后进入平台：
    
    *   **直接弹出注册 / 登录弹窗**
        
*   注册 / 登录成功后：
    
    *   若有用量 → 执行 Screen → 进入详情页
        
    *   若无用量 →  展示升级/购买弹窗
        

![image](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/a/p2003xM1wH85GN9D/2ec485ce5e4942d3923aa2f93d90f56b1049.png)

### 入口 2: 直接 Url 访问平台

#### 未登录用户

*   用户直接访问平台 URL：
    
    *   允许浏览基础页面
        
    *   页面右上角自动展示 Google 帐号注册/登录提示，
        
        *   可参考 [Perplexity](https://www.perplexity.ai/) 右下角页展示轻量注册提示：![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/1wvqrebXPvmKvnak/img/57a966e0-1167-4e80-b416-7222cf55adbe.png) 
            

顶部或右下角轻量注册提示

*   在搜索框点击搜索结果时：
    
    *   在当前页面展示注册/登录弹窗
        
    *   注册/登录完成后，跳转预执行页：
        
        *   有用量：直接 screen 并跳转结果页展示
            
        *   无用量：弹出升级/购买弹窗

### 升级弹窗

升级弹窗中**默认仅展示一个推荐 Plan**。

*   用户可在弹窗中直接选择推荐 Plan，并跳转至 **Checkout 页面** 完成支付方式选择；
    
    *   支付成功后，系统将 **自动执行当前 Screen**，无需用户重复操作；
        
    *   推荐 Plan 会根据用户当前订阅状态动态变化；
        
*   弹窗中始终保留 **View all plans** 入口，供用户查看完整定价方案。
    
    *   如果用户跳转到 Pricing Plan 后完成购买，支付成功也会自动执行当前 Screen，无需要用户重复操作

## 结果分享页优化

结果分享页同样去除遮挡，展示完整的结果数据。

Report Header 文案修改：

*   地址：  
    **You’re viewing a shared KYA report from Phalcon Compliance.**
    

      Results reflect the screen performed on this address at the time it was generated.

*   交易：  
    **You’re viewing a shared KYT report from Phalcon Compliance.**
    

      Results reflect the screen performed on this transaction at the time it was generated.

### 展示策略

*   去除所有遮挡，展示完整结果数据
    
*   数据范围和现在一致，即：
    
    *   Address / Transaction Information
        
    *   Risk Summary
        
    *   Risk Overview
        
    *   Top Counterparties + Transfers （受限，见下）/ Token Transfers
        

### 引导重新扫描

1.   在页面右上角 **Generated at** 时间处提示：
    

*   当前结果为历史快照
    

*   注册后可重新扫描查看最新结果
    

文案：

_This report is a historical snapshot. Screen to view the latest results._

1.  增加固定悬浮提示。
    

**方案一**：在**顶部/ Risk Summary 附近**展示固定的 banner，文案：

:::
This report reflects the **risk status at the time of analysis**.  
New transactions or newly flagged counterparties may introduce additional risk.                                            Check latest risk status **→**
:::

（现在 Risk Summary 的 Avoid xxx 的这句 action 文案可以直接去掉）

This report reflects the risk status at the time of analysis. New activity may change the risk status.

This report is a snapshot. Risk may have changed.                         Check latest risk status →

**方案二**：右侧固定悬浮按钮中的第一个 Sign Up 修改为：Screen latest risks，hover 后展示：Run a new screen to view the latest risk analysis.

### 功能限制与引导

Risk Overview 的资金流图，不提供 MS 资金流图生成功能：

*   `Investigate on MetaSleuth`
    
    *   Hover / Click：
        
        *   引导注册
            
        *   文案：Sign up and re-screen to investigate fund flows on MetaSleuth with full tracing capabilities.
            

Transfer 列表

*   结果分享页中：
    
    *   仅展示最近 **10 条 transfers**
        
*   查看完整 transfers：
    
    *   Hover / Click 引导注册
        
    *   在 transfer 列表上方展示文案：  
        Showing the 10 most recent transfers at the time of the screen. Sign up to view the full transfer history.
