# 用户增长漏斗

**访问 → 注册 → 激活 → 使用 → 付费 → 留存**

说明：

*   **访问**：未登录 & 已登录用户的页面访问
    
*   **注册**：账号创建 / 登录成功
    
*   **激活**：首次完成核心价值行为（Screen）
    
*   **使用**：持续使用核心功能
    
*   **付费**：订阅或购买点数
    
*   **留存**：在后续时间点再次完成核心行为
