# Pricing Model

**Starter Plan**

**$ 149/mo  ($1,499/year)**

*   **150 screens/mo** (addresses & transactions)
    

\-------- 

*   Default Risk Engine (FATF/ HK 选项)
    
*   Export KYA/KYT Report
    

*   Notification Channel: E-mail
    

**Growth Plan**

**$ 299/mo  ($2,999/year)**

*   **320 Screens /mo** 
    

\-------- 

*   Default Risk Engines (FATF/ HK 选项)
    
*   Export STR/SAR Report
    
*   Notification Channel: E-mail; Slack; Telegram; Lark; PageDuty; Discord
    

**Pro Plan**

**$ 699/mo ($6,999/year)**

*   **750 Screens /mo** 
    

\-------- 

*   **API Integration** for automated workflows
    
*   Real Time Transaction Monitoring
    
*   Default Risk Engines (FATF/ HK 选项)
    
*   10 Customizable Risk Engines
    
*   Notification：+ Webhook
    

**Enterprise Plan**

For organizations with large-scale usage and custom requirements.

*   Higher screen volume with **discounted pricing**
