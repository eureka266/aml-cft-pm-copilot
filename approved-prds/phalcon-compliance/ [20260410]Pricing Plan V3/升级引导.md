# 升级引导

> 隶属：Pricing V3 设计文档 · 状态：草稿 V2 功能锁屏和配额 banner 继续复用，本文档仅涵盖 V3 新增场景。

# 一、设计原则

*   **上下文触发**：引导出现在用户刚刚遭遇限制或发现价值的瞬间，不主动打扰
    
*   **直接到位**：从引导到付款只经过一次跳转（Modal 方案直接打开 Checkout Modal）
    
*   **清晰的价值主张**：每条引导说明升级后能解锁什么，而非仅说"升级"
    
*   **不做 Onboarding 弹窗**：不在登录/注册后主动弹窗引导
    

# 二、复用 V2 场景（不重复设计）

*   **功能锁屏**：进入无权限模块时，内容替换为"模块介绍 + 升级 CTA"
    
*   **配额接近上限 banner**：如"You're at 8/10 customers — Upgrade to expand capacity"
    

# 三、V3 新增场景

## 场景 1：Credits 用完后再次尝试筛查

**触发条件：** Credits 用户（无订阅）点击 "Screen" 时账户余额为 0

**弹窗设计：**

```plaintext
┌──────────────────────────────────────────────────┐
│  You've used all your credits             [×]   │
│  ─────────────────────────────────────────────── │
│                                                  │
│  Top up with more credits, or subscribe for      │
│  a better per-screen rate.                       │
│                                                  │
│  ┌────────────────────┐  ┌──────────────────┐   │
│  │  Buy 50 Credits    │  │ Subscribe to      │   │
│  │  $95 · $1.90/scr   │  │ Essential        │   │
│  │  [Buy Credits]     │  │ from $39/mo →    │   │
│  └────────────────────┘  └──────────────────┘   │
│                                                  │
│  Essential gives you 50 screens/mo + STR/SAR    │
│  Reports, Monitoring, Blacklist & more.          │
└──────────────────────────────────────────────────┘
```
> 推荐展示最低档 50 Credits（$95）降低心理门槛；订阅入口展示 Essential 最低档（$39/mo），强调功能价值。

**触发条件：** Free 或 Credits 用户进入 Monitor 功能时，7 天试用已结束（试用资格：账户首次，全局唯一）

**页面提示（嵌入式，替换 Monitor 入口区域）：**

```plaintext
Your free Ongoing Monitoring trial has ended.

Subscribe to Essential to continue monitoring your
addresses in real time.

[Subscribe to Essential →]   [See all plans]
```

**触发条件：** Free 用户输入非 ETH/TRON 地址并点击 Screen

```plaintext
⚠ This chain requires Credits or a subscription.
  Free plan supports ETH & TRON only.
  [Buy Credits]  [Subscribe]
```

## 场景 5：筛查结果低风险 → 引导 Monitor

**触发条件：** 筛查完成，风险等级为 Low 或 Medium

**结果页底部 contextual CTA：**

```plaintext
┌─────────────────────────────────────────────────┐
│  📍 This address looks low-risk now.            │
│  Want to be alerted if it changes?              │
│                                                 │
│  Enable Ongoing Monitoring to track this        │
│  address automatically.                         │
│                                                 │
│  [Monitor this address]  （点击打开 Checkout Modal 或 Add-on 设置）
└─────────────────────────────────────────────────┘
```

*   **Essential/Scale 已有 Monitor 配额**：直接加入监控，不需付费
    
*   **Credits 或 Free 用户**：引导订阅 Essential
    

## 场景 6：筛查结果高风险 → 引导加入 Blacklist

**触发条件：** 筛查完成，风险等级为 High 或 Critical

**结果页底部 contextual CTA：**

# 四、弹窗设计规范

*   **尺寸**：最大宽度 480px，内容短的可用 400px
    
*   **关闭方式**：右上角 × 按钮，点击遮罩关闭（场景 1 为强引导，不支持点击遮罩关闭）
    
*   **主 CTA**：Primary button（蓝色实心）
    
*   **次 CTA**：Ghost button 或文字链
    
*   **场景 1（Credits 用完）**：强引导型，遮罩不可关闭，必须选择操作或点 × 明确关闭
    
*   **其他场景**：软引导型，可随时关闭
    

# 五、Checkout Modal 集成

所有引导场景的付费 CTA 均直接打开 Checkout Modal（参见子文档 3），并预填：

*   目标计划/档位
    
*   若有 Monitor add-on 场景，预选对应 add-on 档位
    

用户点击升级 CTA → 引导弹窗关闭 → Checkout Modal 打开，无页面跳转。
