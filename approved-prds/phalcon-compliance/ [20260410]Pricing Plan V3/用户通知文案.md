# 用户通知文案

> 隶属：Pricing V3 设计文档 · 状态：草稿 上线日期：2026-05-08

# 一、通知对象与时间线

**通知对象：仅付费订阅用户（Starter / Growth / Pro）。Free 用户不通知。**

| 时间节点 | 渠道 | 对象 |
| --- | --- | --- |
| 上线前 7–14 天（约 4 月 24 日 – 5 月 1 日） | 邮件 | Starter / Growth / Pro |
| 上线当天（5 月 8 日） | 邮件 | Starter / Growth / Pro |
| 上线后 30 天内 | 产品内 Banner | 所有已登录用户 |

# 二、变量说明

邮件正文中使用以下占位变量：

| 变量 | 说明 |
| --- | --- |
| `{{first_name}}` | 用户名（无名字时使用 "there"） |
| `{{launch_date}}` | 上线日期，格式：May 8, 2025 |
| `{{old_plan}}` | 旧计划名：Starter / Growth / Pro |
| `{{new_plan}}` | 新计划名：Essential / Scale |
| `{{screening_quota}}` | 月度屏数，如 50 screenings/mo |
| `{{monthly_price}}` | 月付价格，如 $59/mo |
| `{{billing_cycle}}` | monthly / annual |
| `{{renewal_date}}` | 下次续费日期，格式：May 15, 2025 |
