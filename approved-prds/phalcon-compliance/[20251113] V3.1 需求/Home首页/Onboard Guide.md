# Onboard Guide

原onboarding模块占用大面积home首屏空间，现希望折叠进intercom的窗口内，面向不知道如何开始的用户提供试用引导和AI助手问询。

1.  Import your address / Transaction Txn
    
2.  View Risk Details
    
3.  Check Risk Engines and set alert
