# 权限控制及用户引导

**平台访问路径说明**

*   官网导航
    
    *   导航栏登录按钮 → 跳转登录页（已登录用户直接进入对应平台）
        
    *   Compliance产品页搜索框 → 直接进入Compliance平台（免登录）
        
    *   Security产品页操作按钮 → 进入Security平台（需登录验证）
        
    *   其他页面 -> 跳转登录页
        
*   通知渠道Alert消息 → 直达对应平台（未登录时跳转登录页）
    
*   结果分享页面 → 直接进入Compliance平台（免登录）
    
*   管理员平台 →  无所谓
    
*   浏览器直接输入网址：
    
    *   已登录用户 → 进入上次访问的平台
        
    *   未登录用户：
        
        *   曾有Compliance访问记录 → 以Visitor身份进入Compliance
            
        *   无历史记录 → 跳转至登录页
