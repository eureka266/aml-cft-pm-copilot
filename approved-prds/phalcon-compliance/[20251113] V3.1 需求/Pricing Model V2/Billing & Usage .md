# Billing & Usage 

# Usage 页面

Usage 页面用于向所有已登录用户（Free、Starter、Growth、Pro、Enterprise）展示其当前订阅计划下 所有资源的配额上限、已使用量 以及 Plan 限制导致的功能可用性。

页面展示内容：

*   Service Date
    
*   功能用量及权限，如下表
    

| **Feature** | **控制方式** | **Display Format** | **Action** |
| --- | --- | --- | --- |
| **Screening Count** | 周期用量 | `13/50 per month` 或 `240/3000 per year` | Buy Credit (当用户用量耗尽时展示，引导到 Pricing Plan 页面) |
| **Custom Risk Engines** | 限额 | `❌` 或 `1/3 total` | Risk Engine List |
| **Alerts** | Enable / Disable | `✔️` or `❌` | Alert Hub |
| **Analytics** | Enable / Disable | `✔️` or `❌` | Analytics |
| **Customers** | 限额 | `❌` 或 `1/1 total` | Customer List |
| **Labels** | 限额 | `1/3 total` | Label List |
| **Tags** | 限额 | `1/3 total` | Tag List |
| **STR / SAR Report** | Enable / Disable | `✔️` 或 `❌` | \- |
| **Auto Re-screening** | Enable / Disable | `✔️` or `❌` | Screen Settings |
| **Blacklist / Whitelist** | 限额 | `❌` 或 `1/50 total` | Screen Settings |
| **Notification Channels** | 类型 | Email / Telegram / Slack / All | Notification Channels |
| **API Integration** | Enable / Disable | `✔️` or `❌` | API Docs |
| **Seats（Enterprise Only）** | 限额 | `0/0` | Organization |
| **Support（Enterprise Only）** | 限额 | `1/1` | \- |
